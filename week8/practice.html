문제 (Array of Objects)

2차원 평면 위의 점 5개가 주어진다. 좌표를 y좌표가 증가하는 순으로, y좌표가 같으면 x좌표가 증가하는 순서로 정렬한 다음 출력하는 프로그램을 작성하시오.

단, 지난 시간에 배운 클래스의 연산자 오버로딩을 사용하여 문제를 해결하십시오.

입력

5개의 줄에서 차례로 i번째 점의 위치 xi와 yi가 주어진다. (-100,000 ≤ xi, yi ≤ 100,000) 좌표는 항상 정수이고, 위치가 같은 두 점은 없다.

출력

첫째 줄부터 5개의 줄에 점을 정렬한 결과를 출력한다.

예제 입력 1

0 4
1 2
1 -1
2 2
3 3

예제 출력 1

1 -1
1 2
2 2
3 3
0 4

출처

백준 좌표정렬하기2
