Lab6: Grade management system

you would like to write a program that manages the scores for assignments and exams for each student
in your program, there has several operations like table located below.
When you save your student object, you can use vector container.
you can freely implement class functions and variables

We offer you the documentations. Please read them carefully and go to implement.

For any question, make a github issue or contact to TA.

Funtion table

Operation Name Descriptions
add [name] Add students in your program
initiallize all of the score(10 lab/hw, prj, exam) using constructor to 0
setlab [name] [weekN] [score] Enter the [weekN]th lab score of the student corresponding to the name.
there are total of 10 labs from week0 to week9
sethw [name] [weekN] [score] Enter the [weekN]th hw score of the student corresponding to the name.
there are total of 10 homeworks from week0 to week9
setprj [name] [score] Enter the project score of the student corresponding to the name.
setprj [name] [score] Enter the exam score of the student corresponding to the name.
setscoreweight [labratio] [hwratio] [prjratio] [examratio] Determine the weight for each evaluation itmes.
weight is integer type. The sum of given weight should be 10.
fist initialize by [1 2 3 4]
printscore [name] Print the total score of the student corresponding to the name.
printscoredetail [name] Print the score of the student in detail corresponding to the name.
Print all of the lab score, homework score, project score, exam score.

입력

프로그램을 실행하면 operation을 받기 위한 대기 상태가 된다.
operation과 해당하는 input들이 입력되면, 해당 내용을 수행한다.
exit이 입력되기 전까지 계속해서 반복적으로 operation이 수행되며, exit이 입력되면 프로그램이 종료된다.

출력

printscore, printscoredetail에 의한 출력이 나온다.

이전 과제들과 마찬가지로, 출력시 newline이 포함되면 안된다.

예제 입력1

add yuseok
setlab yuseok 0 10
set yuseok 1 9
setprj yuseok 8
setexam yuseok 9
printscore yuseok
printscoredetail yuseok
setgraderatio 1 1 4 4
printscore yuseok
