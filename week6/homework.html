Homework6: POS system

you would like to write a program that manages your market
in your program, thete has several operations like table located below.
you can freely implement class functions and variables

We offer you the documentations and code example. Pleas read them carefully and go to implement.

For any question, make a github issue or contact to TA.

Funtion table

Operation Name Descriptions
buy [item_name] [item_num] [price] [margin] Buy items.
Set the purchased price and margin using constructor.
Take the totally purchased price out of my balance.
If you buy items over your balance, print "can't buy".
When buying the same item, a new price and margin are set. and the item remainings is added.
sell [item_name] [item_num] Sell items.
Change the remaining item number.
Add the value in the balance(selling price is price+margin).
If your item doesn't exist(including out of stock), print "not exist".
If you sell items over your item remainings, print "can't sell".
check [item_name] Print item number(remainings), sell_price(price+margin)
If your item doesn't exist, print "not exist".
balance Print your balance.
exit Print totally earned money.
If there is loss, use "-"
quit your program.

입력

프로그램을 실행하면 int형의 잔고(balance)를 입력받는다.(Market의 총 balance를 의미한다.)
그 후, operation을 받기 위한 대기 상태가 된다.
operation과 해당하는 input들이 입력되면, 해당 내용을 수행한다.
exit이 입력되기 전까지 계속해서 반복적으로 operation이 수행되며, exit이 입력되면 프로그램이 종료된다.

출력

cehck, balance, exit에 의한 출력이 나온다.

이전 과제들과 마찬가지로, 출력시 newline이 포함되면 안된다.

1000000

check bottle
balance

check cup
balance

check basket

exit
