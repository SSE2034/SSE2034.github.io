Homework5: Implementing stringlib.hh

In this homework, you will implement several useful string functions that are not included in the C++ string library.

We offer you the documentations and code example. Pleas read them carefully and go to implement.

For any question, make a github issue or contact to TA.

Important Cautions
  • You only need to code on stringlib.cc, while main.cc are just for testing your code.
  • Do not include other libraries, except that TA previously included.
  • Do not use a function, std::to_string() of <string> library. If you use the function, your code is not graded.
  • When TA grades your codes, new test-case will be added.
Function Name Descriptions
startsWith(string str, string prefix) Returns true if the string str starts with a specific string.
startsWith(string str, char prefix) Returns true if the string str starts with a specific character.
endsWith(string str, string suffix) Returns true if the string str ends with a speicific string.
endsWith(string str, char suffix) Returns true if the string str ends with a speicific character.
integerToString(int n) Converts an integer into the corresponding string of digits.
realToString(double d) Converts a floating-point number into the corresponding string form.
stringToInteger(string str) Converts a string of digits into an integer.
stringToReal(string str) Converts a string representing a real number into its corresponding value.
toLowerCase(string str) Returns a new string in which all uppercase characters have been converted into their lowercase equivalents.
toUpperCase(string str) Returns a new string in which all lowercase characters have been converted into their uppercase equivalents.
compareIgnoreCaseType(string s1, string s2) Returns true if s1 and s2 are equal regardless of the case type.
trim(string str) Return a new vector container which contains each strings of input sentence str.

Function Detail 

bool startsWith(string str, string prefix);
bool startsWith(string str, char prefix);
Returns true if the string str starts with the specified prefix, which may be either a string or a character.
std::string str = "Hello World!";
std::string start_str = "Hel";

if(startsWith(str, start_str))
    std::cout << "true" << std::endl;
else
    std::cout << "false" << std::endl;

// Output: "true"

bool endsWith(string str, string suffix);
bool endsWith(string str, char suffix);
Returns true if the string str ends with the specified suffix, which may be either a string or a character.
std::string str = "Hello World!";
std::string end_str = "rld!";

if(endsWith(str, end_str))
    std::cout << "true" << std::endl;
else
    std::cout << "false" << std::endl;

// Output: "true"

string integerToString(int n);
Converts an integer n into the corresponding string. For example, calling integerToString(123) returns the string "123".
int i = 3291850;

std::string str = integerToString(i);

std::cout << str << std::endl;

// Output: "3291850"

string realToString(double d);
Converts a floating-point number into the corresponding string form. For example, calling realToString(23.45) returns the string "23.45".
double d = 3291.1805

std::string str = realToString(d);

std::cout << str << std::endl;

// Output: "3291.1805" or "3291.1084" etc..
Notice

  • C++에서는 모든 Floating 값들이 오차를 갖지 않는 것은 불가능 합니다. 예를 들어 여러분이 double d = 0.8;를 실제로 변수에는 0.8과는 약간 다른 0.800000001 정도가 할당됩니다. 이는 소수점을 2진수로 표현하려다 발생하는 어쩔수 없는 현상입니다. 따라서 조교는 IEEE-754 Floating Point Converter에서 오차가 발생하지 않는 경우에 한하여 채점을 수행합니다. 예를 들어 13.125 경우에는 오차가 발생하지 않으나 13.126은 오차가 발생합니다. 조교는 13.125를 사용하지 13.126과 같이 오차가 발생하는 숫자들로는 평가하지 않습니다.

  • 자릿수를 지켜주셔야 합니다. 예를들어 18.05를 변환 할때 정확히 string은 "18.05"가 담겨야 합니다. "18.05000"과 같은 것은 감점사항입니다.
int stringToInteger(string str);
Converts a string of digits into an integer. Note that you should automatically cast the real type into integer.
std::string str = "3291850.0000";

int i = stringToInteger(str);

// Output: 3291850

double stringToReal(string str);
Converts a string representing a real number into its corresponding value. 
std::string str = "3211.1855465";

double d = stringToReal(str);

// Output: 3211.1855

Returns a new string in which all uppercase characters have been converted into their lowercase equivalents.
std::string str = "Hello World!!";

std::string new_str = toLowerCase(str);

std::cout << new_str << std::endl;

// Output:"hello world!!"

string toUpperCase(string str);
Returns a new string in which all lowercase characters have been converted into their uppercase equivalents.
std::string str = "Hello World!!";

std::string new_str = toUpperCase(str);

std::cout << new_str << std::endl;

// Output:"HELLO WORLD!!"

bool compareIgnoreCaseType(string s1, string s2);
Returns true if s1 and s2 are equal regardless of the case type.
std::string str1 = "Hello World!!";
std::string str2 = "HELLO WORLD!!";

if(compareIgnoreCaseType(str1,str2))
    std::cout << "true" << std::endl;
else
    std::cout << "false" << std::endl;

// Output:"true"

vector<string> trim(string str);
Divide the sentence by white space and return the divided strings in form of a string container.
std::string str = "Hello!! My name is Chulsoo.";

std::vector<std::string> vec = trim(str);

for(int i=0; i<vec.size(); i++){
    std::cout << vec[i] << std::endl;
}

// Output:
// Hello!!
// My
// name
// is
// Chulsoo.
