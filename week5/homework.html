stringlib.h

This file exports several useful string functions that are not included in the C++ string library.

Function Name Descriptions
startsWith(string str, string prefix) Returns true if the string str starts with a specific string.
startsWith(string str, char prefix) Returns true if the string str starts with a specific character.
endsWith(string str, string suffix) Returns true if the string str ends with a speicific string.
endsWith(string str, char suffix) Returns true if the string str ends with a speicific character.
integerToString(int n) Converts an integer into the corresponding string of digits.
realToString(double d) Converts a floating-point number into the corresponding string form.
stringToInteger(string str) Converts a string of digits into an integer.
stringToReal(string str) Converts a string representing a real number into its corresponding value.
toLowerCase(string str) Returns a new string in which all uppercase characters have been converted into their lowercase equivalents.
toUpperCase(string str) Returns a new string in which all lowercase characters have been converted into their uppercase equivalents.
comapreIgnoreCaseType(string s1, string s2) Returns true if s1 and s2 are equal regardless of the case type.
trim(string str) Return a new vector container which contains each strings of input sentence str.
