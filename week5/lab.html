Lab5: Implementing matrixlib.hh

In this lab, you will implement several useful functions that are not included in the C++ library.

This library perfroms simple matrix operations for only NxN square matrix(정사각 행렬).

We offer you the documentations and code example. Pleas read them carefully and go to implement.

For any question, make a github issue or contact to TA.

Read Me
  • You only need to code on matrixlib.cc, while main.cc are just for testing your code.
  • The size of matrix is defined at param.hh. You must make a code that can operate even if the constant value of N is changed.
  • When TA grades your codes, all the testing code will be changed. For examle, the size of square matrix will differ and new test-case will be added.
Function Name Descriptions
matEqual(int A[][N], int B[][N]) Returns true if two matrix contain the same elements.
matAdd(int A[][N], int B[][N], int C[][N]) Perform add operation between matrix A and B.
matSub(int A[][N], int B[][N], int C[][N]) Perform subtraction operation between matrix A and B.
matMul(int A[][N], int B[][N], int C[][N]) Perform multiplication operation between matrix A and B.
transpose(int A[][N], int A_T[][N]) Perform transportation of matrix A.
matrixToString(int A[][N]) Return a string converted from matrix array.

Function Detail 

bool matEqual(int A[][N], int B[][N]);
Returns true if two matrix contain the same elements.
int A[2][2] = {{1,2},{3,4}};
int B[2][2] = {{1,2},{3,4}};

if(matEqual(A,B))
    std::cout << "Equal" << std::endl;    
else
    std::cout << "Not equal" << std::endl;

// Output: Equal

void matAdd(int A[][N], int B[][N], int C[][N]);

C = A + B

Perform add operation between matrix A and B.

And transfer the result to matrix C.

int A[2][2] = {{1,2},{3,4}};
int B[2][2] = {{1,2},{3,4}};
int C[2][2] = {0,};

matAdd(A, B, C);

// Now C={{2,4},{6,8}}

void matSub(int A[][N], int B[][N], int C[][N]);

C = A - B

Perform subtraction operation between matrix A and B.

And transfer the result to matrix C.

int A[2][2] = {{1,2},{3,4}};
int B[2][2] = {{1,2},{4,8}};
int C[2][2] = {0,};

matSub(A, B, C);

// Now C={{0,0},{-1,-4}}

void matMul(int A[][N], int B[][N], int C[][N]);

C = A.dot(B)

Perform multiplication operation between matrix A and B.

And transfer the result to matrix C.

int A[2][2] = {{1,2},{3,4}};
int B[2][2] = {{1,2},{3,4}};
int C[2][2] = {0,}

matMul(A, B, C);

// Now C = {{7, 10}, {15, 22}}

void transpose(int A[][N], int A_T[][N]);

A_T = A.transpose()

Perform transportation of matrix A.

And transfer the result to matrix A_T.

int A[2][2] = {{1,2},{3,4}};
int A_T[2][2] = {0,}

transpose(A, A_T);

// Now A_T = {{1, 3}, {2, 4}}

string matrixToString(int A[][N]);

Return a string converted from matrix array.

int A[2][2] = {{1,2},{3,4}};

string str = matrixToString(A);

std::cout << str << std::endl;

// Output:
// {1,2},
// {3,4}
Notice

  • Output:은 출력 대상이 아닙니다.

  • 오해의 소지가 있어서 큰따음표(")를 삭제했습니다. 큰따음표(")는 string에 포함될 대상이 아닙니다.

  • 중괄호({), 콤마(,), 엔터키(\n) 모두 string에 들어갈 대상입니다. 예시와 똑같이 출력해주세요.
